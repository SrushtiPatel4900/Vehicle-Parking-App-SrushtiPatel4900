# vehicle-parking-app
Vehicle Parking App - V1 is a Flask-based web application for managing 4-wheeler parking lots.
It supports two roles: Admin and User.
Admins can create/edit parking lots, view spot statuses, and manage users.
Users can register, book parking spots, and release them.
The system auto-allocates spots and tracks parking time and cost.
Built using Flask, SQLite, HTML/CSS, Bootstrap, and Chart.js.
